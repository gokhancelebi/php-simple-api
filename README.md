# PHP SIMPLE API

This is a simple API framework for PHP. It is based on [REST](http://en.wikipedia.org/wiki/Representational_state_transfer) and [JSON](http://en.wikipedia.org/wiki/JSON).

## Features

 * Routing
 * Class mapping
 * Cache (Coming soon)
